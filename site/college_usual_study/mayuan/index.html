    速通方法

    • 平时分：预置的是汪建达老师。老师虽然评分很低，但是事情较少，只有一篇读书报告还有一个课堂展示。发言一次加五分，上限两次，这个还是很好拿的，老师提问非常多，慢慢ai也来得及。唯一缺点是不能翘课,老师每节课都会抽人回答问题，没来按照旷课处理。论文交的是打印稿，所以感觉要求也不高。
    • 期末考：倍速听彭双贞老师的复习课，每年都是差不多的，如果时间太晚可以听去年前年的。老师思路非常清晰，语音也很清晰，非常适合ai总结。
    • 大题：花一些时间把psz老师讲的大题背熟了，内容非常少，记性好的应该1h可以背熟。（我记性不好少食多餐了好几天）
    • 小题：把98上搜罗到的回忆卷一次一次喂给ai，开始是分析每一个错题，然后记住之后再让他出题，反复搞了几次，比较简单的题目可以做出来。
    • 考场：选择题拿下做过的，其他就按照这样复习培养出的一点点“马性”乱蒙。大题一定要多写，写满，真诚才是必杀技。我其实背的很少，又没有选考政治，大题是写不出有营养的东西的，但是就是一句话掰成好几句，结合点时事热点，再自己随便胡编乱造，字写大一点，硬是写满了答题纸，奋战到了最后一刻。

    笔记整理

    大题

    总纲领

    • 分点依次全部默写。（不需要挑选正确的相关的，一股脑塞上）
    • 每个点后面，把材料内容转述成自己的话，按照知识点分析。我一般是把材料的内容写哪里体现哪里，又再把知识点替换成材料里面具体的名字写，凑字数。
    • 写满答题纸。

    唯物论

    1. （物质观）世界是物质的，物质决定意识，要尊重客观规律/一切从实际出发。

    2. （意识观）意识具有能动性。

    3. （统一观）要坚持主观能动性与客观规律性的统一。

    唯物辩证法

    1. （总特征） 唯物辩证法认为，要以联系和发展的眼光看问题。

    2. （对立统一规律 — 核心，分三小点）

      • 2.1 矛盾具有统一性和斗争性，要一分为二地看问题。

      • 2.2 矛盾具有普遍性和特殊性，要具体问题具体分析。

      • 2.3 矛盾有主要方面和次要方面、主要矛盾和次要矛盾，要坚持两点论和重点论的统一。

    3. （质量互变规律） 量变和质变是辩证统一的，既要注重量的积累，又要适时推动质变。

    4. （否定之否定规律） 事物的发展是否定之否定的过程，是前进性和曲折性的统一。

    认识论

    1. （实践决定认识） 实践在认识中起决定作用。

    2. （认识的本质） 认识的本质是主体在实践基础上对客体的能动反映。

    3. （认识的过程） 认识过程是一个从实践到认识、再从认识到实践的无限发展过程。

    历史唯物主义

    1. （社会历史观基本问题） 社会存在决定社会意识，社会意识反作用于社会存在。

    2. （社会基本矛盾） 生产力决定生产关系，经济基础决定上层建筑；生产关系对生产力、上层建筑对经济基础具有反作用。

    3. （群众史观） 人民群众是历史的创造者。

    4. 如果材料内容与科学技术相关，则必须额外加上这句话：“科学技术是社会发展的重要动力，要正确把握科学技术的社会作用。” 并可就其“双刃剑”效应进行阐述。

    政治经济学两大基础原理

    1. 劳动价值论：具体劳动创造商品的使用价值，抽象劳动形成商品的价值。

    2. 剩余价值论：揭示了资本家通过无偿占有劳动者的剩余价值（剩余劳动时间），以实现资本增值的过程。

    3. 使用场景：当材料涉及 商品生产、价值来源、工资与利润、劳资关系 时，可先陈述这两个原理，再结合材料分析。

    4. 考试就凭这句话颠来倒去写满了整页纸，，，出分后比预期高很多很多，多写还是有用的

    经济危机

    • 核心观点（必背两句）：

      • 经济危机的实质是 “生产相对过剩”。

      • 经济危机的根源是 “资本主义基本矛盾”，即生产社会化与生产资料资本主义私人占有之间的矛盾。

    • 答题逻辑：指出实质与根源 → 结合材料说明（如材料中的产品积压、消费不足等现象） → 阐述此矛盾如何导致危机。

    国家垄断资本主义

    核心定义（必背）：国家政权与私人垄断资本的融合。

    • 答题逻辑（“是什么-为什么-影响”框架）：

      1. 是什么：先给出上述定义。

      2. 为什么：简要说明其形成原因（如应对危机、战争需要、加强干预等）。

      3. 表现形式：可提及国家管制、财政调节、国有企业等（无需死记，点到即止）。

      4. 影响：辩证分析。既有缓解矛盾、促进发展的积极作用；也有无法克服基本矛盾、可能加剧社会不公的消极面。

    经济全球化及其挑战（如 逆全球化）

    核心立场（必背）：经济全球化是不可逆转的历史趋势，逆全球化现象只是暂时的逆流。

    • 答题逻辑：

      1. 肯定全球化是生产力发展的必然结果。

      2. 分析材料中全球化带来的积极影响（如资源优化配置）与负面问题（如发展失衡）。

      3. 针对“逆全球化”材料，指出其实质是保护主义，违背历史潮流。应致力于推动更加包容、普惠、共赢的经济全球化。

    当代资本主义新变化

    答题逻辑（灵活框架）： - 承认当代资本主义在生产力、生产关系、上层建筑各方面出现新变化（如数字经济、福利制度等）。

    - 运用 **“资本主义基本矛盾”** 这一根本原理进行分析：这些**新变化并未触动资本主义私有制的根本，因而无法最终克服其基本矛盾**。

- 结论：资本主义的自我调节是有限度的，其内在矛盾最终会以新的形式表现出来。

    导论

    一、 马克思主义的产生背景

    1. 阶级基础（实践基础）：

      • 无产阶级作为独立的政治力量登上历史舞台。

      • 三大工人运动/起义：

        • 法国里昂纺织工人起义。

        • 英国宪章运动。

        • 德国西里西亚纺织工人起义。

    2. 思想渊源（直接理论来源）：

      • 对应马克思主义三大组成部分：

        • 德国古典哲学 → 马克思主义哲学。

        • 英国古典政治经济学 → 马克思主义政治经济学。

        • 英法空想社会主义 → 科学社会主义。

    3. 自然科学前提：

      • 19世纪自然科学的三大发现：

        • 细胞学说。

        • 能量守恒与转化定律。

        • 生物进化论。

    二、 马恩的思想转变与重要著作

    1. “两个转变”的完成：

      • 马克思和恩格斯发表在 《德法年鉴》 上的论文，标志着他们完成了：

        • 从唯心主义向唯物主义的转变。

        • 从革命民主主义向共产主义的转变。

    2. 标志性著作及其意义：

    著作名称 核心意义 / 标志
    《神圣家族》 马克思和恩格斯第一次合写的著作。
    《关于费尔巴哈的提纲》 被恩格斯称为 “包含新世界观的天才萌芽的第一个文件”。其中最后一句：“哲学家们只是用不同的方式解释世界，问题在于改变世界。”
    《德意志意识形态》 首次系统阐述历史唯物主义基本原理，实现了历史观上的伟大变革。（提出“社会存在决定社会意识”）
    → 马克思的第一个伟大发现：唯物史观
    《共产党宣言》 标志马克思主义的公开问世。同时也是科学社会主义诞生的标志。
    《资本论》 系统阐述剩余价值学说，被誉为 “工人阶级的圣经”。
    → 马克思的第二个伟大发现：剩余价值学说
    《反杜林论》 恩格斯撰写，全面系统地阐述了马克思主义理论体系，马克思主义三大组成部分的框架源于此。

    1. 关键理论关联：

      • 马克思一生的两个伟大发现：唯物史观 和 剩余价值学说。

      • 社会主义从空想变为科学的理论基石：正是基于以上两大发现。

    三、 马克思主义的鲜明特征

    1. 本质属性：人民性。马克思主义是人民的理论，脱离人民性就不是马克思主义。

    2. 核心观点：实践观点是马克思主义首要的和基本的观点。

    3. 理论品质：与时俱进。马克思主义是发展的、开放的理论。

    4. 活的灵魂：具体问题具体分析。（反对教条主义和经验主义）

    5. 基本特征（一句话概括）：科学性与革命性的统一。

      • （注：人民性、实践性、发展性都内在地统一于、体现为革命性。）

    6. 世界观与方法论基础：辩证唯物主义和历史唯物主义。（注意：此处是哲学基础，不同于“两大发现”是理论基石）

    三、 "最值"、"核心"、"本质"篇（填空题杀手）

    【马克思主义总论】

    1. 马克思一生的两大伟大发现：唯物史观 + 剩余价值学说。
    2. 马克思主义最鲜明的品格 / 本质属性：人民性（为什么人服务）。
    3. 马克思主义首要的、基本的观点：实践观点。
    4. 马克思主义的活的灵魂：具体问题具体分析。
    5. 根本方法：辩证唯物主义和历史唯物主义（如矛盾分析法）。

    【哲学·辩证法】

    1. 唯物辩证法的总特征：联系 + 发展。
    2. 唯物辩证法的实质与核心：对立统一规律（矛盾规律）。
    3. 事物发展的动力和源泉：矛盾（对立统一）。
    4. 区分事物的依据：矛盾的特殊性。
    5. 割裂共性与个性（普遍性与特殊性）的例子：公孙龙"白马非马"。

    【哲学·认识论】

    1. 意识产生的决定性环节：劳动。
    2. 检验真理的唯一标准：实践。
    3. 实践区别于认识的特点：直接现实性（把想的变成真的）。
    4. 认识的来源/动力/目的：实践。
    5. 真理的属性：客观性（内容客观）、绝对性、相对性。

    【哲学·唯物史观】

    1. 社会发展的决定性力量：生产方式（生产力+生产关系）。
    2. 区分社会经济时代的客观依据：生产工具（生产力水平）。
    3. 区分社会经济结构（社会性质）的依据：生产资料所有制。
    4. 社会历史观的基本问题：社会存在 vs 社会意识。
    四、 政治经济学逻辑篇（死逻辑，不能乱）

    1. 价值与劳动

    • 价值的唯一源泉：抽象劳动（活劳动）。
    • 具体劳动创造：使用价值。
    • 抽象劳动创造：价值。
    • 商品的二因素：使用价值 + 价值。

    2. 货币与资本

    • 货币的本质：一般等价物。
    • 资本积累的唯一源泉：剩余价值。（剥削来的钱不花，攒下来做本钱）。
    • 扩大再生产的源泉：资本积累。

    3. 剩余价值的生产（剥削方式）

    • 绝对剩余价值：延长时间（加班）、提高强度（累死你，不让休息）。
    • 相对剩余价值：提高全社会劳动生产率（技术进步），导致生活资料便宜，缩短必要劳动时间。

    4. 资本的划分

    • 不变资本 (C)：机器、厂房、原材料。（不创造新价值，只转移价值）。
    • 可变资本 (V)：工人/劳动力。（可以创造出比自身工资更多的新价值）。

    5. 经济危机

    • 根本原因：资本主义的基本矛盾（生产社会化 vs 生产资料私人占有）。
    • 直接原因/本质：生产相对过剩。
    五、 那些必须记住的"坑"
    1. 静止与运动：物质是运动的承担者；静止是运动的衡量尺度。
    2. 理想与现实：只要符合规律，理想总能转化为现实。（不符合规律的是空想）。
    3. 自由王国：指共产主义社会（摆脱了盲目必然性，人成为主人）。
    4. 圣经 vs 出生证：《资本论》是圣经；《宣言》是出生证。
